## About the project

For this non-profit organization we developed an informational website
where visitors can find an overview of all events organized by the organization.

The website makes it easy to discover upcoming events, browse photos from
previous editions and find more information about the organization’s activities.

### Key improvements

- clear overview of upcoming events
- photo gallery of previous events
- fast and mobile-friendly website
- central place for information and updates
