## About the project

For this project we developed an online ticket platform where organizations
in the event industry can publish their events and sell tickets.

Event organizers get access to a management dashboard where they can easily
create events, configure ticket types and track ticket sales.

### Key improvements

- manage events through a dedicated dashboard
- multiple ticket types per event
- real-time overview of sold tickets
- secure online payments via Stripe
